## 📌 Project Scope

This project visualizes sales data across multiple dimensions for FY 2025–26. It compares year-over-year performance, breaks down revenue by category and region, and highlights top-performing products with month-on-month changes.

---

## 📊 Features

- Monthly Sales Trend — FY 25–26 vs FY 24–25 (Line Chart)
- Category Revenue Share (Doughnut Chart)
- Quarterly Performance Comparison (Bar Chart)
- Region-wise Sales (Horizontal Bar Chart)
- Top Products Table with MoM % Change
